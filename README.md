JBucks – Personal Expense Management System

JBucks is a simple and intuitive web application to track daily expenses. It allows users to record their own spending as well as amounts paid on behalf of others. The system provides clear monthly summaries, category-wise breakdowns, and a dedicated view showing how much has been paid for each person.

Key Features

Add expenses under categories like Food, Traveling, Entertainment, and Other

Record “Paid for Others” with person name, amount, description, and date

Automatic creation and listing of payees

Monthly dashboard showing:

Total spent by you

Total spent for others

Category-wise totals

Per-person summary page showing total amount paid for each person

Edit and delete expense entries

Responsive UI (mobile + desktop)

Hosted live using Flask + PostgreSQL + Render

Tech Stack

Frontend: HTML, CSS (Bootstrap), JavaScript

Backend: Python, Flask

Database: SQLite (local), PostgreSQL (production)

Deployment: Render
